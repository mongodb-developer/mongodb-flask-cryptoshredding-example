Welcome!
This demo provides an interface to signup a new user, which will trigger the generation of a new data encryption key in the background. You can then login with this user, and start saving some data into the database. We will store the data like this, in a collection named data: 
            
{
  "username": <your username (encrypted)>
  "key": <the key you enter>,
  "value": <the value you enter (encrypted)>
}
Visit the admin page, where you can trigger a bunch of different actions to test - for example deleting the data encryption key for your user. To get started, sign up with a user.
